# Public docs live under docs-public

Public Documentation is stored in `docs-public/`, while Internal Documentation
stays in `docs/`. Public URLs continue to use
`https://diagrampilot.com/docs/...`, so the repository path is intentionally not
mirrored in the hosted URL path.

`llms.txt` links only Public Documentation. Maintainer workflow docs, ADRs,
roadmaps, and project process notes remain internal and are discovered through
repository guidance such as `AGENTS.md`.
